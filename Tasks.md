## Tasks.md（具体步骤拆解）

> 目标：创建一个与分享稿 `definations.md` 对应的 7 章节互动演示 Demo（Streamlit + Pandas + Plotly），并保证可一键启动与可演示。

### 0. 环境与脚手架
- **T0.1 准备 Python 环境**
  - **操作**：安装 Python 3.8+；确保 `python3` / `pip3` 可用
  - **验收**：`python3 --version` 正常输出
- **T0.2 初始化项目目录**
  - **操作**：创建目录 `260122-vibe_coding分享2.0/`，新增 `.streamlit/`
  - **验收**：目录结构可被识别（后续文件都放到位）
- **T0.3 写依赖清单 `requirements.txt`**
  - **操作**：写入 `streamlit>=1.28.0`、`pandas>=2.0.0`、`plotly>=5.17.0`
  - **验收**：`pip3 install -r requirements.txt` 安装成功
- **T0.4 写 Streamlit 配置 `.streamlit/config.toml`**
  - **操作**：配置 `address=0.0.0.0`、`port=8501`、关闭 usage stats
  - **验收**：启动后监听 `8501`，且允许局域网访问（如需要）
- **T0.5 写启动脚本 `start.sh`**
  - **操作**：检测 `python3`/`streamlit`，缺失则安装依赖后 `streamlit run ai_coding_demo.py`
  - **验收**：执行 `./start.sh` 能拉起应用

### 1. 应用骨架（先跑起来）
- **T1.1 创建主程序 `ai_coding_demo.py`**
  - **操作**：引入 `streamlit/pandas/plotly.graph_objects`；调用 `st.set_page_config(layout="wide")`
  - **验收**：`streamlit run ai_coding_demo.py` 能打开页面且无报错
- **T1.2 注入全局 CSS**
  - **操作**：用 `st.markdown(..., unsafe_allow_html=True)` 定义 `.highlight-box/.warning-box/.success-box/.quote-box`
  - **验收**：页面盒子样式可见且一致
- **T1.3 侧边栏导航路由**
  - **操作**：`st.sidebar.radio("选择章节", [7个标题])` 产出 `page`
  - **验收**：切换不同章节会进入不同渲染分支（先用占位标题即可）
