# 快速开始

## 🚀 一键启动（推荐）

```bash
cd "/Users/boning/Desktop/beike_cursor/260122-vibe_coding分享2.0"
./start.sh
```


应用将在 http://localhost:8502 启动

## 📋 手动启动

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 启动应用
streamlit run ai_coding_demo.py
```

## ❓ 常见问题

**Q: 端口被占用怎么办？**

编辑 `.streamlit/config.toml`，修改端口号：
```toml
[server]
port = 你的端口号
```

**Q: 依赖安装失败？**

使用国内镜像源：
```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q: 如何停止应用？**

在终端按 `Ctrl+C`
